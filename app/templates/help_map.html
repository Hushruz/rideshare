How do I create a new event? How can I ask for a ride? How can I drive a ride? How can I create a ride to an event?
Create a new event by clicking 'Create a new Event' in the top panel. You will be asked to select an event location and supply a few additional details. After your event is created, you can view it by visiting the Events page.
You can ask for a ride by clicking 'Create a new Ride' in the top panel. You will be asked for the starting location of the ride and then select you are requesting a ride.
It is possible someone is offering a ride to where you want to go. Examine the map and the Rides page to find out.
You can create a ride by clicking 'Create a new Ride' in the top panel. You will be asked for the starting location of the ride and then select you will be the driver.
You can create a ride to an event by finding the event on the map. Events are denoted by blue icons. Click on the icon and you will have the option to create a ride to that event.
